Инсталиране на gtk библиотека чрез следната команда:
============================================================
sudo apt-get install build-essential libgtkmm-2.4-dev -y

Компилиране чрез следната команда:
------------------------------------------------------------
g++ file_in.cpp -o file_out `pkg-config --libs --cflags gtkmm-2.4`

Страница на бибиотеката:
------------------------
https://www.gtkmm.org/en/
